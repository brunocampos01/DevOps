# How to README?

Precisa ter o seguinte formato: 
- Title 
- Tags<br/>
Procure por tags aqui.
- Describe project 
- Pre Requirements
- Quickstart
- Install
- Examples
- References 
- Bug fix or sugestions
- How to contribuite </br>
(explicação de como enviar pull requests, ajudo a escrever isso) 
- Acthor
- Copyright and license
